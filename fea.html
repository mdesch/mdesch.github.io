This Project is a Work In Progress
The Objective of this project is to create a javascript backed finite element analysis tool for K-12 Educational Purposes. The Stiffness method will be used to calculate structural response to point and/or disributed loads. User input of structure geometry to be implemented. Define: Nodes, Members, Material Properties, Section Properties, Point Loads, Distributed Load, Return: Internal Forces, Reactions, Deflections
