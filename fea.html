Finite Emement Procedure

1.) Define Materials
2.) Define Cross sections
3.) Define Nodes
4.) Define Elements
5.) Define Bounrady Conditions
6.) Define Forces

*Don't name multiple materials the same thing

*Don't name multiple sections the same thing

*Reccomended to assign (0,0) as the lower-leftmost point of the structure

*Don't choose the same node at both the start and end of the element

*Only assign one boundary condition per node, or it will only consider the last b.c. applied

*Only apply one (x,y) force pair per node, or it will only consider the last force pair assigned
